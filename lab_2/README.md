# Лабораторная работа: Бинарная классификация танков (СССР vs Германия)

## Описание задачи

Реализована бинарная классификация изображений танков на две категории: **советские** и **немецкие**.  
Датасет включает **31 изображение**, разделённое на два уровня сложности:

- **Лёгкая сложность**: реальные фотографии танков.
- **Тяжёлая сложность**: кадры из мультфильмов, игрушки, детские рисунки, чертежи, памятные монеты и др.

## Технические детали

- **API-ключ Hugging Face** хранится в файле `.env` (для запуска требуется вставить ваш ключ).
- Для сравнения использованы **image-text-to-text модели**:
  - **Qwen**
  - **Gemma**
- Ссылки на изображения находятся в файле [`urls.csv`](urls.csv).
- Промпт для моделей:  
  _"Write one word: Soviet if the picture shows a Soviet tank, German if the picture shows a German tank."_

## Методология

1. Реализована функция классификации, принимающая:
   - Ссылки на изображения
   - Название модели
   - API-ключ
   - Текст промпта
2. Проведено три независимых запуска классификации для каждой модели.
3. Рассчитаны метрики качества и проанализированы типы ошибок.

## Результаты

| Модель    | Accuracy (среднее) | Особенности ошибок                                                                                                                 |
| --------- | ------------------ | ---------------------------------------------------------------------------------------------------------------------------------- |
| **Gemma** | ~0.93              | Повторяющиеся ошибки на:<br>- Памятнике советскому танку в Чехии<br>- Чертежах                                                     |
| **Qwen**  | ~0.92              | Ошибки на:<br>- Чертежах (всегда)<br>- 1 случайная ошибка на изображениях лёгкой сложности (СССР/Германия)<br>- Фотографии игрушек |

## Выводы

- Обе модели показали **сопоставимую точность** (~93%), но с разными паттернами ошибок.
- **Gemma** демонстрирует устойчивость к вариациям.
- **Qwen** чувствительна к сложным изображениям (чертежи, игрушки).
- Среднее время классификации составило около нескольких секунд на изображение.
- Обе модели можно успешно использовать для поставленных задач.
